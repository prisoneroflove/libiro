## Prerequisite
  * A C /C++ Compiler (GCC,Clang...)
  * Cmake, Make

## Building  
  
  * Clone the repo
  * Move to the src directory
  ```
  cd src
  ```
* Run Cmake
  ```
  cmake CMakeLists.txt
  ```
* Run Make
  ```
  make
  ```
* Run the file
  > For Linux/MacOS
  ```
  ./libiro
  ```
  > For Windows
   ```
  .\libiro
  ```

   * To clean
  ```
  make clean
  ```
